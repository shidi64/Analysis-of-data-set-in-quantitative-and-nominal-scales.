1) Количественные данные 
1. Сбор в интернете набор данных о ставках рефинансирования и курсах 
национальных валют России и Казахстана не менее, чем за последние 50 месяцев 
(как минимум одно наблюдение на месяц). Курсы валют должны быть приведены к 
одинаковым единицам измерения.
2. Загрузка собранной информации в набор Pandas DataFrame. 
3. Графическое исследование для каждой из переменных на предмет 
совпадения её распределения с нормальным при помощи графиков P-P и Q-Q. Стандатизация данных.
4. Проверяем приблизительное равенство моды, медианы и среднего арифметического 
значения исследуемого набора данных. Строим их на гистограмме исследуемого 
набора данных. 
5. Проводим аналитическое исследование для каждой из переменных на предмет 
совпадения её распределения с нормальным. Обоснованно выбераем один из 
методов: Шапиро-Уилка, Андерсона-Дарлинга или Колмоговора-Смирнова. 
6. Удаляем наиболее заметные выбросы и повторите исследование. 
7. Для каждой из пар переменных графическое исследование данной пары 
на предмет взаимной корреляции. 
8. Для каждой из пар переменных выполняем оценку с помощью методов Пирсона, 
Спирмена, Кендалла. Объясните, какой из методов было лучше использовать в 
вашем случае. 
2) Номинальные данные 
1. На основе загруженного CSV-файла создаем Pandas DataFrame, подобрав 
правильные типы данных столбцов.  
2. Создаем новый Pandas DataFrame, выбрав только переменные EmploymentField, 
EmploymentStatus, Gender, LanguageAtHome, JobWherePref, SchoolDegree, Income. 
3. Удаляем все наблюдения, содержащие либо значения поля пол (Gender), отличные 
от male или female, либо значения NA (нет ответа) в каких-либо из полей. 
4. Исследуем связи между парами переменных (используйте только наблюдения, где 
эти поля заполнены):  
a) Gender, SchoolDegree; 
b) Gender, JobWherePref; 
c) JobWherePref, LanguageAtHome; 
d) EmploymentField, LanguageAtHome; 
e) EmploymentStatus, LanguageAtHome. 
Для каждой пары  строим таблицу сопряжённости, таблицу ожидаемых значений. Обоснованно выбераем один из методов: хи-квадрат Пирсона, хи-квадрат Пирсона с поправкой  Йейтса,
точный критерий Фишера (обычный или на основе приближения Монте Карло), точный критерий Фримана-Холтона (обычный или на основе приближения Монте-Карло).
